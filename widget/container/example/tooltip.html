Tooltip Example

The creation of a Tooltip is very straight-forward. To instantiate a Tooltip, the code looks like this:

tooltip1 = new YAHOO.widget.Tooltip("tt1", { context:"btn1", text:"I'm a button. You can click me!" } );

Tooltips are generally always created dynamically, and never need to be marked up in advance. The first argument is the DOM id that you ant to associate with the Tooltip. The second argument is the user configuration object literal used for configuring the Tooltip. If no text is specified, the Tooltip will attempt to use the object's title attribute to set the text.

By default, are shown after 2 seconds, displayed for 5 seconds and dismissed, or hidden 250ms after moving the mouse away from the context element. You can configure these timings, as shown in the code for tooltip3 (the one associated with the image), where we cause the Tooltip to display after 1 second, instead of 2:

tooltip3 = new YAHOO.widget.Tooltip("tt3", { context:"img", showdelay: 1000 });
