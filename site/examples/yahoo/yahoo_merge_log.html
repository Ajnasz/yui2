YAHOO Global Object: Managing simple data sets with YAHOO.lang.merge

YAHOO.lang.merge allows you to combine any number of input hashes and output a single merged set. This example demonstrates combining multiple data sets.

Note the result will contain keys from all supplied sets and any keys present in multiple sets will store the value from the last set.

var result = YAHOO.lang.merge( 
{
  value   : '',
  size    : 20,
  max     : -1,
  auto    : false,
  calendar: false

}
, 
{
  value   : 'Foo',


  auto    : true,

  validate: true
}
, 
{

  size    : 30,
  max     : 100,
  auto    : false,
  calendar: true

}
);

result

{
  value   : 'Foo',
  size    : 30,
  max     : 100,
  auto    : false,
  calendar: true,
  validate: true
}

The example: merging three hashes

In the example illustrated above, we merge three data hashes that share some keys. Note the value of 'auto' is false in the result because key values in later data hashes trump those in hashes before them. Also, the key:value pair for 'validate' is included in the result even though it was not present in prior hashes.

Merge any number of sets

You can pass as many parameters to merge as you'd like. The result starts as an empty object and grows from there.

Priority to the last

Each argument passed to merge has its key:value pairs added to a result object. The returned result will contain all keys from all objects passed. If two or more sets contain the same key, the value from the latter set will be assigned to the result.

Shallow merges

merge operates at the root level of each object. Nested structures will not have their keys merged as well.

Merge does not alter the inputs

The objects passed to merge are examined, not modified. Use YAHOO.lang.augmentObject to modify objects in place.
