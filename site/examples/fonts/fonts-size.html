Fonts CSS: Setting the font size

Using YUI Fonts CSS, specify font size using the provided percentages. (For example, 77% renders at 10px.) By using these percentages you get accuracty across A-grade browsers while still allowing IE users to zoom their font size.

View example in new window.

About This Example

With YUI Fonts CSS in place, define font sizes using the percentages found in the documentation and on the cheatsheet. For example, 77% renders as 10px; 138.5% renders as 18px.

Because CSS examples are susceptible to other CSS on the page, this example is only available in a new window at the above link.
