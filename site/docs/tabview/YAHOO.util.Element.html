Class YAHOO.util.Element - uses YAHOO.util.AttributeProvider

Element provides an interface to an HTMLElement's attributes and common methods. Other commonly used attributes are added as well.

available
Fires when the Element's HTMLElement can be retrieved by Id.

See: Element.addListener

Event fields:
