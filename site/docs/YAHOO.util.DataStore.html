setItem

Boolean setItem ( item , location )
Saves data to local storage. It returns a String that can be one of three values: "true" if the storage succeeded; "false" if the user has denied storage on their machine, and "pending" if storage is permitted, but the storage space allotted is not sufficient.

The size limit for the passed parameters is ~40Kb.

Parameters:
item <Object> The data to store
location <String> The name of the "cookie" or store
Returns: Boolean
Whether or not the save was successful

setSize

void setSize ( value )
This method requests more storage if the amount is above 100KB. (e.g., if the store() method returns "pending". The request dialog has to be displayed within the Flash player itself so the SWF it is called from must be visible and at least 215px x 138px in size.
Parameters:
value <object> The size, in KB
Returns: void
